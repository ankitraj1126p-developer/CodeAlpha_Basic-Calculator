# 🧮 Basic Calculator Website 

📌 Project Title

 Basic Calculator Website

A modern calculator web application built using HTML, CSS, and JavaScript.  
This project allows users to perform basic arithmetic operations with both button clicks and keyboard input support.


 ✨ Features

⌨️ Keyboard Support  

Type numbers and operators directly using keyboard and press **Enter** to calculate result.

🧮 Basic Arithmetic Operations  

Supports addition (+), subtraction (−), multiplication (×), and division (÷).

🧹 Clear Function (AC)  
Clears the entire display instantly.

⌫ Delete Function  
Deletes the last entered character.

📊 Percentage Calculation  
Supports percentage (%) operations.

🔢 Bracket Support  
Handles opening and closing brackets automatically.

🎨 Responsive UI Design 

Modern calculator interface with styled buttons and layout.

 🛠️ Technologies Used

HTML5 – Structure of the calculator  
CSS3 – Styling and layout  
JavaScript – Calculator logic and keyboard support  
Font Awesome – Icons (Delete button)
